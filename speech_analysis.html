Overview

  • This language learning tool utilizes speech recognition and speech-to-text technology, combined with sentiment analysis from TextBlob, to provide users with feedback on the polarity and subjectivity of their recorded audio
  • The tool features a Tkinter-based GUI, allowing users to easily start and stop recordings
  • Feedback is delivered in both text and audio formats through the GUI, enhancing the learning experience

Technologies Used

  • Python, pyttsx3, TextBlob, speech_recognition, tkinter

Key Contributions

  • Developed modules for speech recognition, sentiment analysis, and text-to-speech feedback
  • Designed a user-friendly GUI for real-time interaction
  • Employed modular programming practices

Achievements

  • Fully functional speech analysis application with real-time feedback
  • Enhanced user engagement through auditory feedback
