Overview

  • PDF Sentiment Analysis using pdfplumber
    • This program reads text from any imported PDF file and performs sentiment analysis using the VADER Sentiment Analyzer from the NLTK library. After running the script, the sentiment analysis results will be displayed in GUI window.
  • URL Sentiment Analysis using BeautifulSoup
    • This program reads text from the contents within any url and performs sentiment analysis using the VADER Sentiment Analyzer from the NLTK library. After running the script, the sentiment analysis results will be displayed in GUI window.
